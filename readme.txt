
-> CREDITS

This is the first public release of the CBM-II IEC cartridge.

The code is made available by Michal Pleban. For the latest version, look here:
http://www.cbm-ii.com/

IEC routines and machine language monitor were taken from Ullrich von Bassevitz
code, available at:
ftp://ftp.musoftware.de/pub/uz/cbm610/

-> INSTALLING

The CART.BIN file is a binary image which should be burned into a 2764 EPROM.
This EPROM should be placed in a cartridge slot at $6000

-> COMPILING

The source can be compiled using CA65 assembler from the CC65 package.
